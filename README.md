An eCommerce store for fashion lovers üòç

## **Local Setup**

```bash
$ git clone https://github.com/arp99/plovex-store.git
$ cd plovex-store
$ npm install
```

## **Technologies Used**

- React
- Redux Toolkit
- React Router v6
- Tailwind Css, Emotion, Twin.macro
- MongoDB
- Mongoose
- ExpressJS
- JWT token based Authentication

## **Functionalities**

- Home page Showcasing brands, categories and new releases
- Individual product page
- Cart
- Wishlist
- Filter and sort products
- Authentication:
  - User Signup
  - User Login
